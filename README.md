# 📊 E-Commerce Sales Dashboard

An interactive Power BI dashboard built to analyze e-commerce sales data and transform raw transactions into actionable business insights.

## 🚀 Project Overview

This project demonstrates the complete Power BI workflow, including data cleaning, data modeling, DAX calculations, and dashboard development. Using e-commerce sales data, the dashboard helps analyze sales performance, profitability, customer behavior, payment preferences, and regional trends.

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* CSV Files

## 📂 Dataset

The project uses two datasets:

* **Orders.csv** – Order details, dates, cities, states and payment modes
* **Details.csv** – Product categories, quantities and sales amount

## 📈 Dashboard Highlights

* Total Sales Analysis
* Profit Analysis
* Average Order Value (AOV)
* Category & Sub-Category Performance
* State-wise Sales & Profit Analysis
* Payment Mode Distribution
* Monthly Sales & Profit Trends
* Financial Year Comparison
